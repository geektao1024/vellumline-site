# Vellum & Line Final Logo Assets

Selected direction: **Vellum Monogram**.

The final mark uses a refined `V` shape as the vellum form and a warm orange memory line as the route/place trace. It is intended for the current dark UI and project assets.

## Final Files

- `svg/vellum-line-monogram.svg`: master symbol.
- `svg/vellum-line-monogram-lockup.svg`: horizontal lockup.
- `svg/vellum-line-monogram-app-icon.svg`: PWA/app icon source.
- `png/vellum-line-monogram-1024.png`: large transparent symbol.
- `png/vellum-line-monogram-512.png`: medium transparent symbol.
- `png/vellum-line-monogram-256.png`: small transparent symbol.
- `png/vellum-line-monogram-lockup-2048.png`: transparent lockup.
- `png/vellum-line-app-icon-512.png`: dark app icon.
- `png/vellum-line-og-banner-1200x630.png`: social preview banner.
