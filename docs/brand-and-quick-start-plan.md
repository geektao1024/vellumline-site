# Vellum & Line Brand And Quick Start Plan

日期：2026-05-27

## 1. 品牌基线

- 产品名：`Vellum & Line`
- 域名：`https://vellumline.app`
- 产品类型：custom map poster maker / printable map art studio
- 主 CTA：`Open Studio`
- 工具入口：`/app`

一句话定位：

Vellum & Line 是一个浏览器里的地图海报工作台，用来把真实地点、城市纹理、路线轨迹和纪念文字做成可导出的地图海报。

## 2. 受众

- 想做城市、家乡、旅行、纪念日地图海报的用户。
- 想把跑步、骑行、自驾、徒步路线做成墙面作品的用户。
- 想找 Terraink 风格地图海报工具、但希望流程更直接的搜索用户。
- 需要快速生成可打印地图艺术图的设计师、创作者和礼物购买者。

## 3. 语气

推荐表达：

- `custom map poster maker`
- `printable map poster`
- `meaningful places`
- `route poster`
- `browser-based map studio`
- `export-ready map art`
- `OpenStreetMap-based map design`

避免表达：

- 泛泛的模板站、增长、自动化、全能平台话术。
- “官方 Terraink 替代品”这类可能造成误解的表达。
- 没有依据的排名、最好、第一、唯一等绝对化说法。
- 与地图海报无关的模板产品、案例或功能说明。

## 4. 页面优先级

第一优先级：

- 首页
- `/app` 入口
- Terraink 替代页
- Terraink 对比页
- CTA
- FAQ
- 导航和 footer

第二优先级：

- 价格页
- 案例页
- 博客页
- 更新页
- Legal 页面

## 5. 首页规则

首页要同时承担两个任务：

1. 让用户马上知道 Vellum & Line 是地图海报工具。
2. 承接 Terraink 相关搜索，但不能把首页写成附属说明页。

推荐结构：

- Hero：产品名 + custom map poster maker + 打开工作台。
- 使用流程：搜索地点、调整风格、加入路线或文字、导出海报。
- 场景：城市海报、路线海报、婚礼/旅行/礼物海报。
- Terraink Guide：说明独立产品关系，并给比较页入口。
- CTA：进入 `/app`。
- FAQ：围绕导出、路线、打印、Terraink 关系、数据来源。

## 6. SEO 关键词

首页主词：

- `custom map poster maker`
- `map poster maker`
- `Terraink`
- `Terraink alternative`

扩展词：

- `printable map poster`
- `route poster maker`
- `running route poster`
- `wedding map poster`
- `city map poster`
- `OpenStreetMap poster`

## 7. 视觉规则

- 默认暗黑主题。
- 不显示主题切换。
- 标题尽量居中，版块节奏保持一致。
- 图标使用统一线性风格。
- 图片必须与地图、路线、海报、工作台相关。
- 不使用无关产品截图、泛 SaaS 插画或模板案例。

## 8. 验收清单

- 公开页面不出现模板品牌词。
- 公开页面不出现无关模板案例。
- 导航、CTA、footer 都指向 Vellum & Line 的真实路径。
- Terraink 文案明确独立关系。
- `/app` 能从顶部导航进入。
- 中英文页面均可访问。
- 本地 `pnpm exec tsc --noEmit` 通过。
